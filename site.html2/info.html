O que é um Servidor FTP?

FTP (File Transfer Protocol) é um protocolo de rede para transferência de arquivos entre sistemas.

O que é JavaScript?

JavaScript é uma linguagem de programação usada para criar interatividade em páginas web.

O que é Docker?

Docker é uma plataforma para desenvolvimento, envio e execução de aplicações em containers.
